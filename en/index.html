OSP2-BE

Quake 3 Arena mod

Extended edition of the OSP2.
OSP2 is an attempt to recover the old OSP mod source code and improve upon it.
Based on vanilla Q3 and OSP files analysis.

OSP2: ModDB - Source code - by Snems
OSP2-BE: ModDB - Source code - by diwoc Send email
Community: OSP2 Chat - OSP2-BE News
Acknowledgements
Support Project

The mod requires original OSP 1.03 resources or compatible files to work properly.

  1. Download the required .pk3 file above
  2. Download original OSP 1.03 or OSP 1.03 resources
  3. Place all files in the osp folder in your Quake 3 Arena directory

For other mods use the appropriate folder (or any other folder of your choice):

  • efreeze for Q3Retro
  • unfreeze for FPSClassico

Note: OSP2 also requires original OSP 1.03 resources to work correctly.
