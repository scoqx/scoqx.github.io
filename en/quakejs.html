QuakeJS

Play Quake 3 in your browser

QuakeJS is a port of ioquake3 to JavaScript using Emscripten, allowing you to play Quake 3 Arena directly in your web browser.
Experience the classic Quake 3 gameplay with OSP2-BE mod support in your browser.

Features

  • Full Quake 3 Arena gameplay in browser
  • OSP2-BE mod support
  • Multiplayer support via WebSockets
  • No installation required
  • Cross-platform compatibility

How to Play

  1. Click the "Launch QuakeJS" button below
  2. Wait for the game to load (may take a few minutes on first visit)
  3. Use your mouse and keyboard to play
  4. Press F11 for fullscreen mode

Controls

  • WASD - Movement
  • Mouse - Look around
  • Space - Jump
  • Ctrl - Crouch
  • Shift - Run
  • 1-9 - Weapon selection
  • Enter - Chat
  • Esc - Menu

QuakeJS Information

System Requirements

  • Modern web browser (Chrome, Firefox, Safari, Edge)
  • JavaScript enabled
  • WebGL support
  • Stable internet connection
  • At least 2GB RAM recommended

Technical Details

  • Based on ioquake3 engine
  • Compiled with Emscripten
  • WebSocket networking
  • WebGL rendering
  • Web Audio API for sound

Known Issues

  • Initial load time may be slow
  • Some audio features may not work on all browsers
  • Performance depends on browser and hardware
  • Fullscreen mode requires F11 key
